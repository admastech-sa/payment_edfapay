EdfaPay Payment Gateway

Secure, Fast, and Reliable Payment Processing for Odoo 19

Overview

Integrate the EdfaPay payment gateway into your Odoo 19 environment. Provide your customers with a reliable, secure, and swift checkout experience directly on your Odoo eCommerce store. This integration seamlessly bridges your Odoo sales modules with EdfaPay, ensuring optimized transaction processing, automatic payment validation, and real-time status updates.

💡 How It Works

  1. Customers browse your eCommerce store and proceed to checkout.
  2. They select EdfaPay as their preferred payment provider.
  3. The system securely processes the transaction via the EdfaPay payment portal.
  4. Upon success, the order is confirmed and the payment status is immediately validated in Odoo.

🛠️ Configuration

  1. Install the module from your Odoo Apps database.
  2. Navigate to Accounting > Configuration > Payment Providers.
  3. Select EdfaPay from the provider list.
  4. Input your EdfaPay Merchant Key and API credentials.
  5. Set the state to Production and publish to activate the payment gateway online.
